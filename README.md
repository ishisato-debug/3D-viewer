# 3D Viewer for GLB Files

Blenderで作成した3Dオブジェクト（.glb形式）をWebブラウザで観察できるツールです。

## 特徴

- 📱 PC/スマホ両対応
- 🎯 GLB形式に対応
- 🔄 全方向から自由に観察可能
- ⚡ ローカルファイルなので素早く読み込み

## 使い方

1. [ビューアーを開く](https://ishsato-debug.github.io/3d-viewer/index.html)
2. 「ファイル選択」からBlenderで出力した`.glb`ファイルを選択
3. マウスやタッチで自由に操作

### 操作方法

**PC:**
- ドラッグ: 回転
- スクロール: ズーム
- 右クリックドラッグ: パン移動

**スマホ:**
- ドラッグ: 回転
- 2本指ピンチ: ズーム

## Blenderからのエクスポート

1. Blenderでオブジェクトを選択
2. File → Export as → glTF Binary (.glb)
3. ファイルを出力

その後、このビューアーで開けます。# 3D-viewer
